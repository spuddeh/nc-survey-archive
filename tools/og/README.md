# OG card builder

Regenerates the social-share image at `assets/img/og-image.jpg` (1200×630).

The card is an HTML template (`og.html`) rendered in headless Edge so it uses
the **real site fonts** — Night Corp Display (local `NightCorpDisplay-Regular.woff2`),
Fira Code + Rajdhani (Google Fonts) — and theme colors, matching the website
exactly. (ffmpeg's text renderer segfaults on Fira Code, so compositing text
directly won't work — hence the browser.)

## Files
- `og.html` — the card layout (text, sizes, colors, gradient).
- `NightCorpDisplay-Regular.woff2` — brand title font (copy of `assets/fonts/`).
- `bg.jpg` — the background crop (regenerated by `build.sh`).
- `build.sh` — regenerate the card end to end.

## Regenerate
```bash
bash tools/og/build.sh [background-image]
```
- `background-image` is optional; defaults to a Kabuki rooftop capture in
  `uploads/`. Pass any frame to use a different backdrop.
- Then **bump the `?v=` query** on `og:image` / `twitter:image` in `index.html`
  (e.g. `?v=3` → `?v=4`) so social validators (and their caches) re-fetch the
  new image, and commit + push. Pages redeploys automatically.

## Edit the design
Change the text / fonts / sizes / gradient in `og.html`, preview it by opening
it through a local server (`cd tools/og && python -m http.server`), then run
`build.sh` to write the JPEG.

Requires: `ffmpeg`, `python`, Microsoft Edge (Windows / Git Bash).
